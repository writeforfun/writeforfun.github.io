json筆記

發表於 2021-03-19 分類於 Web ， data 閱讀次數：

因為在設定hexo時必須修改一堆.yml檔，而剛好yml跟json很像，所以就做了這篇簡短的筆記。

簡介

json (JavaScript Object Notation)，唸法同Jason，跟yaml一樣都是key-value寫法。
差別在:

  • 換行的時候json要加逗號而yaml不用
  • json的key要加雙引號

寫法

  • 可用縮排把上述資料包進person這object裡面
    • 呼叫方式: person.gpa
  • []裡面可以存:
    • 單變數像hobbies: [xx, xx, xx]
    • 多變數像friends: [{xx, xx}, {xx, xx}]
  • 記得要逗號~
